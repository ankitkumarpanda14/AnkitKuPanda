Covid Cases Tracking

March 2020

Technologies Used: Selenium, BeautifulSoup4, Android, Requests
Languages used: Python, Java
Github Link: Covid Tracker App

The project aims to make the tracking of the number of active cases for the current day easier to access and also at the same time shows the number of increased cases from the last point of record. The data is taken from the gov india website for displaying the covid data. It shows the data in the form of a graph for better visualization of the data. The Github repo is given in the link above.
