Water Quality Monitoring

A real time system developed from scratch using Raspberry Pi and sensor modules to monitor the water quality in real time and display the data collected using an android application with forecasting the future values of pollutants using Machine Learning Techniques. Also, sends warning to user if water threat levels are high.

Air Quality Prediction

A research project carried out as a part of my thesis for graduation, it involves cleaning unprocessed data of different urban city locations, performing and comparing different Nan filling techniques and predicting the future values of the air pollutant with the maximum accuracy.

Real Time Attendance system

This project involved creating a fingerprint and RFID module for the attendance to be taken with the ID cards of the students or the fingerprint of the students. The cards helped to log the details of the student to an excel sheet and as per the time of arrival, the attendance was marked as present or absent for the day.
