Air Quality Prediction

January 2020 - Present

Technologies Used: RNNs, LSTMs, CNNs, SARIMA, TCN, FbProphet, Android, TCN, Firebase
Languages used: Python, Java

This is a research project that I am currently involved in where I am forecasting time series data for the future using various types of Deep Learning and Machine Learning Models with Regression techniques. The data being used is filled with missing values hence, those also need to be handled in a proper way with imputation techniques such as MICE, KNN Imputer. Overall I am currently working on producing the results with least value of error metrics such as RMSE, MAPE, MAE. Alongwith the data science aspect to it I have also included user usability by developing my own android application which can display the data analysis on the app and at the same time give a warning to the user if in case the AQI level in the users area is above a certain threshold.

Since, the work here is involved in research and have to be published before sharing owing to plagiarism concerns. I will be adding to Github as and when my work gets published or gets a recognition.
