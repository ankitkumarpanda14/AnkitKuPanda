Water Quality Management

January 2019 - March 2019

Technologies Used: IoT, FbProphet, Android, RaspberryPi, Firebase
Languages used: Python, Java, C++
Github Link: Waqman

This project was carried out by me in my third year of study as a part of product development. The project involves collecting data from turbidity and pH sensors using Raspberry Pi and Arduino and then storing the data on the Firebase real time database which will be retrieved using an android application to display the data. Furthermore, the future predictions were done for these values using FbProphet.
